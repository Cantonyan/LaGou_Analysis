探索“数据分析师”岗位¶

简介¶

探索“数据分析师”这个岗位所需要的技能。

首先抓取拉钩网上的数据，再对数据进行清洗，最后对清洗后的数据进行分析。

数据分析包括：探索对于不同工作年限和不同薪酬，“数据分析师”岗位所要求的技能是如何变化的，通过分析结果规划“数据分析师”岗位的学习路线。

还对岗位进行线性回归，找出对薪酬影响最大的因素；也展示了使用statsmodles、sklearn进行回归的能力。

1.数据抓取和清洗¶

1.1 在Get_JobData.py中，限定广州区域，搜索“数据分析”关键字，抓取各个岗位的信息，包括公司名字、职位、岗位要求和薪酬等信息；

1.2 在Data_Wrangling.py中，对抓取的数据进行清洗，将一列的工作年限转换成多列，将薪酬单位由千元转换成元；

1.3 在Data_Wrangling.py中，还使用了jieba对岗位要求进行分词，根据分词结果判断该岗位对技能是如何要求的。

def Sort_Skill(df):
    Job_num = len(df)
    df_skill = df.iloc[:,0:13] #删除年份和薪酬
    Label = list(df_skill.axes[1]) #提取标签
    SkillMat = np.array(df_skill) #并转换成矩阵
    SkillSum = SkillMat.sum(axis=0) #列方向相加
    SkillNum_sort = sorted(SkillSum, reverse=True) #从大到小排序
    Index_sort = np.argsort(-SkillSum) #获取排序后的标签顺序
    Labels_sort = [] #排序后的标签
    for i in range(13): 
            Label_sort = Label[Index_sort[i]]
            Labels_sort.append(Label_sort) #排序后的标签
    return Job_num, SkillNum_sort, Labels_sort

def Draw_Skill(Job_num, SkillNum_sort, Labels_sort, Title):
    plt.rcParams['font.sans-serif'] = ['simHei'] 
    plt.rcParams['axes.unicode_minus'] = False
    plt.figure(figsize=(10,5), dpi = 100) #设定图片宽度和精度
    ax1=plt.subplot(111) #设定绘图范围
    width=0.5 #设定条形宽度
    x_bar=np.arange(13)
    rect=ax1.bar(x=x_bar,height=SkillNum_sort,width=width,color="lightblue") #绘图 
    for rec in rect: #添加数据标签
        x=rec.get_x()
        height=rec.get_height()
        ax1.text(x+0.15,1.02*height,str(int(height)))

    ax1.set_xticks(x_bar) 
    ax1.set_xticklabels(Labels_sort) #X轴坐标
    ax1.set_ylabel("岗位数(共%d个岗位）" %(Job_num)) #y轴坐标
    ax1.set_title(Title) #标题
    plt.show()

总结：
1、通过对“数据分析师”岗位的分析，最需要掌握的工具为SQL，其次为编程语言Python和R，在数据分析师生涯的各个阶段都需要使用。
2、“数据分析师”在初级阶段时还需要掌握使用Excel，中级时需要掌握统计学软件SPSS、SAS，高级时需要掌握Hadoop。
